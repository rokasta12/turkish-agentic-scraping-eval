# Security Policy

This is a research/evaluation harness, not a production scraping system.

Do not use it to bypass access controls, CAPTCHAs, paywalls, account bans, login walls, or robots.txt restrictions.

Default safety rules:

- Use public unauthenticated pages only.
- Respect robots.txt and conservative rate limits.
- Do not extract personal/private data.
- Do not submit forms.
- Do not perform destructive or public mutation actions.
- Do not use authenticated browser profiles for crawling.

If you find a security issue, do not open a public issue with sensitive data.
Contact the maintainer privately.
